# simpCAD

好像还没有正经写过README。

顾名思义，写一些想说的话，想招着手喊“READ ME！”的话，应该还是合适的。

本次编写这个小程序，一拿到题目的时候我是有点崩溃的。这……超纲了吧？开个玩笑。首先是再认真读了下作业的要求，看了看范例视频，列出了要实现的功能：

1. 绘制图形（直线、长方形、椭圆、文字）
2. 选中图形
3. 改变图形特性（大小、颜色、位置）
4. 删除图形
5. 保存和恢复

嗯，就此看来，图形是一个父类/接口，具体形状是其子类/实现了接口的类。每种图形都有可更改的大小、颜色、位置属性。

画布和按钮可以用课上讲过的JPanel和JButton。窗口就用JFrame。

选中图形功能的实现需要在JPanel上AddMouseListener。改变图形特性，如果用视频里那样的按键控制的话，要AddKeyListener。

保存和恢复略讲过，不过在图形界面上调用好像还缺一点什么……

好在关于GUI的东西，看了Java官方的教程之后明白了很多。强烈感谢！

我最纠结的是图形虚类（我最终用了一个叫Figure的Interface）和具体图形之间的关系……到底是接口好还是抽象类好，两边都导致一定程度的代码重复。

最后我还是选了接口，把平面几何留给自带的类了。
